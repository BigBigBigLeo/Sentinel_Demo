# Sentinel 60s QA Acceptance Checklist

## Playback Context A: Silent Pitch Playback
- `PASS` if viewers can restate "proactive defense + ROI uplift" after one watch.
- `PASS` if Chinese subtitles alone carry complete narrative continuity.

## Playback Context B: Meeting Room With Audio
- `PASS` if English VO and Chinese subtitles remain semantically aligned cue-by-cue.
- `PASS` if subtitles do not occlude critical chart/UI regions.

## Playback Context C: Local Laptop 1080p
- `PASS` if playback is smooth at 25fps and text remains sharp.
- `PASS` if no visible subtitle jitter or dropped frames in transitions.

## Playback Context D: Website Hero Embedding
- `PASS` if core "proactive defense" message is clearly delivered within first 15 seconds.
- `PASS` if CTA is legible at standard desktop viewport.

## Playback Context E: Compliance Review
- `PASS` if KPI values are clearly labeled as demo placeholder ranges.
- `PASS` if no unsupported absolute business claims are present.
